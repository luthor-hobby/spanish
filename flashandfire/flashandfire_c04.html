recuerdo falsificadofalsified memory — falsificar = to forge, to falsify
huella dactilarfingerprint — dactilar from dáctilo = finger (Greek root)
irradiando inexistenciaradiating nonexistence — striking phrase for his psychic camouflage ability
anclaanchor — metaphorical: the only lead he has
gramática
  • Si Thess tuviese un espectáculo — Imperfect subjunctive in si-clause: 'if Thess had a show.' Contrary-to-fact condition uses imperfect subjunctive + conditional.
  • No hay vuelta atrás — Fixed expression: 'there's no going back,' literally 'there is no return back.' Very common idiom.
